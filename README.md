Sneaker Mode is a comprehensive e-commerce website developed as part of a web development course project. 
The project was divided into three parts over four months, each building upon the previous to create a fully functional, responsive, and user-friendly sneaker e-commerce platform.

Built 6 complete HTML pages

Content strategy implementation
